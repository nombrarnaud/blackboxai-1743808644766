# BBQ Event Reservation

## Project Overview

The BBQ Event Reservation project is a web application designed to help users create and manage reservations for BBQ events. Leveraging modern web technologies like Vue.js and Tailwind CSS, the application offers a user-friendly interface for participants to RSVP to events, making it easier to organize gatherings.

## Installation

To set up the project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```bash
   cd bbq-event-reservation
   ```

3. **Open `index.html` in your preferred web browser**. The project does not require additional installations since it is a front-end project using CDN links.

## Usage

1. Open the `index.html` file in a web browser to see the BBQ Event Reservation app in action.
2. Interact with the application by filling in the reservation details (the detailed functionality may be defined in `src/main.js`, which is not provided here).

## Features

- User-friendly interface for event reservation.
- Responsive design utilizing Tailwind CSS for modern aesthetics.
- Vue.js for reactive data handling and component-driven architecture.
- Font Awesome icons to enhance visual appeal and usability.

## Dependencies

This project utilizes the following dependencies hosted via CDN:

- **Vue.js**: A progressive JavaScript framework for building user interfaces. [Vue.js CDN](https://unpkg.com/vue@3.3.4/dist/vue.esm-browser.js)
- **Tailwind CSS**: A utility-first CSS framework for creating custom designs.
- **Font Awesome**: A library for scalable vector icons.

The above dependencies are included in the `index.html` file.
